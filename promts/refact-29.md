Измени функцию AnalyzeProject следующим образом:
- функция должна возвращать массив строк.
- первый элемент массива должен быть названием проекта
- следующие элементы массива должны быть именами расширений конфигурации 1С
Для этого измени g.AnalyzeProject следующим образом:
- функция должна возвращать массив строк
- первый элемент массива должен быть названием проекта
- следующие элементы массива должны быть именами расширений конфигурации 1С
- измени логику поиска названия проекта дополнив случаем когда расширений конфигурации нет (тоесть нет каталогов <Название проекта>.<Имя расширения конфигурации>)
  - в этом случае название проекта должно быть равно названию единственного каталога не начинающегося с точки
  - если таких каталогов несколько, то возвращать ошибку и пустой массив строк
