Реализуй в config.go функцию GetConfigData, которая будет получать массив байтов из файла ресположенного в репозитории gitea.
При этом:
- если имя файла начинается с точки (.) то данные получаются из секрета пользователя указанного в константе GitOpsSystemUser. Имя секрета равно имени файла без точки

- если имя файла не начинается с точки то данные получаются из файла находяшегося в корне текущего репозитория. Имя репозитория берется из cfg.GiteaURL+cfg.Owner+cfg.Repo

- если имя файла начинается с подчеркивания (_) то данные получаются из переменной пользователя указанного в константе GitOpsSystemUser. Имя переменной равно имени файла без подчеркивания

если данные не удалось получить или полученное значение пустое - возвращается ошибка
