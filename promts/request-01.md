Есть приложение на go которое работает в контейнере. В дальнейшем планируется работа приложенния вне контейнера на windows. Приложение должно работать на windows и linux, при этом работа на linux является обязательным требованием, работа на windows - опциональным.
Приложение сохраняет все данные на git сервере и не требует сохранения данных на диск.
В процессе работы приложение клонирует большие репозитории (200000+ файлов, 8 ГБ+ объем ) во временные каталоги на диске. Производит с помощью данных из этих репозирориев различные операции и сохданяет результаты на git сервер. Часть операций с данными репозиториев производятся сторонними программами.
