# Задача
Расширить существующий проект на Go новым функционалом.

## Требования
- В файл **main.go** необходимо добавить новую команду.
- Эта команда должна запускать `1cv8` с параметрами:
  - `ENTERPRISE`
  - остальные параметры — такие же, как у других команд для работы с базами 1С
  - `/Execute <путь к временному файлу>`

## Логика работы
1. Получить параметр `StartEpf`, переданный через **action.yaml**.
2. По этому параметру (полный URL) скачать файл из репозитория.
3. Сохранить его во временный файл.
4. Запустить `1cv8`, передав путь к этому временному файлу в параметре `/Execute`.

## Ограничения и рекомендации
- Максимально использовать уже существующие функции и общий стиль кода проекта.
- Поддерживать единообразие с другими командами и функциями в проекте.
- Реализовать решение так, чтобы оно легко расширялось и сопровождалось.
