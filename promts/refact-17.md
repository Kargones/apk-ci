Создай команду которая:
- Получает из переданных параметров полный путь к репозиторию и токен доступа.
- получает список корневых каталогов ветки main репозитория не начинающихся с точки.
- находит имя каталога не включающее точку.
- Если есть хотя бы один каталог такого вида <найденное имя>.<произвольная строка> то:
-- найденное имя считается именем проекта.
-- все строки после точки считаются именами расширений
- выводит информацию о имени проекта и список расширений.
Добавь эту команду в main.go switch cfg.Command.
