доработай документ docs/smb-copy-architecture.md с учетом следующих изменений:
- запуск должен производиться только с использованием механизма action из модуля main.go 
- конфигурация должна производиться только с использованием файла app.yaml
- пароли должны загружаться из Config.SecretConfig.Passwords.Smb
- все остальные параметры конфигурации должны загружаться из Config.AppConfig.Smb
- тесты должны позволять вводить необходимые данные в структурах теста
