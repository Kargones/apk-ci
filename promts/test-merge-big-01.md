При выполнении операции по выявлению конфликтов слияния (TestMerge) с большими репозиториями вместо ответа есть или нет конфликт gitea API возвращает сообщение о том что операция будет продолжаться в фоне.
Модифицируй функцию таким образом чтобы она дожидалась окончания проверки конфликтов слияния в gitea и возврашала корректный результат для больших репозиториев в синхронном варианте, то есть дожидалась окончания проверки и возвращала однозначный вариант.
