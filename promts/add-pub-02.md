Измени логигу работы функции FindSubscribedRepos на следующую:
- Вызывающий код вызывает функцию FindSubscribedRepos(logger, api, sourceRepo)
- FindSubscribedRepos инициализирует логгер
- FindSubscribedRepos инициализирует кэш orgReposCache и slice subscribers
- FindSubscribedRepos получает список названий веток для поиска подписчиков на основании шаблона 
  -  `{org}_{repo}_{extDir}` где 
    - `org` - организация в Gitea текущего репозитория
    - `repo` - имя текущего репозитория
    - `extDir` - директория расширения. Имена расширений получаются с помощью функции AnalyzeProject
- Функция FindSubscribedRepos делает запрос к Gitea API для получения списка подписчиков репозитория
  - Для этого получаем список всех репозиториев во всех организациях к которым удается получить доступ с помощью токена доступа
  - Для каждого репозитория проверяем существует ли в нем ветка с именем `{org}_{repo}_{extDir}`
    - Если ветка существует, то добавляем репозиторий в список подписчиков
- Gitea API возвращает список подписчиков репозитория
- Функция FindSubscribedRepos возвращает список подписчиков репозитория и nil
