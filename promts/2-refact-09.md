Создай модуль entity/filer который реализует следующий функционал:
- работу с файловой системой, при этом файлы могут размещаться как на диске так и в файловой системе в памяти.
- реализует интерфейс io/fs
- при создании экземпляра модуля можно указать путь к файловой системе на диске.
- если путь к файловой системе на диске не указан, то файловая система будет создана каталоге из константы filer.DefaultPath.
- если при создании передается параметр MemoryFS, то файловая система будет создана в памяти.
- при создании файловой системы в памяти она должна быть оптимизирована для работы с большим количеством текстовых файлов маленького размера.
- файловая система в памяти должна быть потокобезопасной.
- файловая система в памяти должна поддерживать команду удаления файловой системы с возращением освобожденной памяти OS.
- файловая система в памяти предназначена для временных операций и не должна быть использована для постоянного хранения данных.
- файловая система в памяти должна быть освобождена после использования.
- файловая система в памяти должна поддерживать работу в linux и windows.
