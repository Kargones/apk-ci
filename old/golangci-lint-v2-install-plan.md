# План установки golangci-lint версии 2

## Текущее состояние проекта

1. В проекте уже есть скрипт установки golangci-lint: `scripts/inst-golangci-lint.sh`
2. Существующий скрипт устанавливает последнюю версию golangci-lint
3. В проекте есть конфигурационный файл `.golangci.yml`, настроенный для работы с версией 2
4. Проект использует Go версии 1.24.3
5. Есть Makefile с командой lint для запуска линтера

## Требования для установки golangci-lint версии 2

1. Необходимо установить конкретно версию 2.x golangci-lint
2. Скрипт должен быть совместим с различными операционными системами (Linux, macOS, Windows)
3. Скрипт должен проверять, установлена ли уже нужная версия
4. Скрипт должен устанавливать golangci-lint в GOPATH/bin или другую подходящую директорию

## План реализации

1. Создать новый скрипт `scripts/inst-golangci-lint-v2.sh`, который будет устанавливать golangci-lint версии 2
2. Модифицировать существующий скрипт `scripts/inst-golangci-lint.sh` для установки версии 2 вместо последней версии
3. Обновить документацию по установке инструментов разработки
4. Проверить работоспособность скрипта установки

## Содержимое нового скрипта inst-golangci-lint-v2.sh

Скрипт должен:
1. Определять операционную систему и архитектуру
2. Скачивать конкретную версию 2.x golangci-lint с GitHub
3. Устанавливать её в $(go env GOPATH)/bin
4. Проверять установленную версию

## Команда установки для версии 2

Для установки golangci-lint версии 2 можно использовать одну из следующих команд:
1. `go install github.com/golangci-lint/cmd/golangci-lint@v2.x`
2. Скачать бинарный файл с GitHub и установить вручную

## Рекомендуемый подход

Рекомендуется использовать первый подход с `go install`, так как он:
1. Автоматически обрабатывает зависимости
2. Корректно работает с GOPATH
3. Проще в обслуживании
