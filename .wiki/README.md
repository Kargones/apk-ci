## О проекте

benadis-runner - это инструмент автоматизации для работы с системой 1C:Enterprise, предоставляющий модули для:

- Конвертации данных между форматами
- Восстановления и управления базами данных MSSQL
- Управления сервисным режимом информационных баз 1C
- Интеграции с 1C:Enterprise Development Tools (EDT)
- Централизованного управления конфигурацией

## Технический стек

- **Язык программирования**: Go (1.25.0)
- **Основные библиотеки**: 
  - github.com/denisenkom/go-mssqldb (подключение к MSSQL)
  - github.com/ilyakaznacheev/cleanenv (парсинг конфигурации из переменных окружения)
  - gopkg.in/yaml.v3 (обработка YAML)
- **Тестирование**: github.com/stretchr/testify

## Быстрый старт

```bash
# Сборка
make build

# Запуск тестов
make test

# Просмотр справки
./build/benadis-runner --help
```
